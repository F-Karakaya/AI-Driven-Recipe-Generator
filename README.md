# AI Driven Recipe Generator

## Description
AI-Driven-Recipe-Generator is an AI-powered recipe generator that helps users create recipes with ease. Users simply type in the name of a dish, and AI-Driven-Recipe-Generator provides ingredients, instructions, an image of the dish, and a fictional story related to it.
